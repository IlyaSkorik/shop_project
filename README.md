# Django Shop Project — Картины по номерам

## Описание проекта
Данный проект является учебной лабораторной работой по разработке веб-приложения на фреймворке Django. Основная цель — освоение навыков развертывания окружения, настройки маршрутизации и работы с базовыми представлениями (views).

Тематика магазина базируется на ассортименте [Leonardo Hobby](https://leonardohobby.by) и специализируется на продаже **товар для подарков на праздники**.

## Функционал
На данном этапе приложение представляет собой каркас магазина с настроенной маршрутизацией и текстовыми страницами.

### Список страниц:
1.  **Главная страница (`/`)** — точка входа, содержащая навигационные ссылки на разделы «Об авторе» и «О магазине».
2.  **О магазине (`/about/`)** — информация о тематике проекта.
3.  **Об авторе (`/author/`)** — страница с данными о разработчике, выполнившем работу.
4.  **Каталог (`/ishop/`)** — страница с категориями каталога таваров.
5.  **Магазины (`/shop/geo/minsk/`)** — страница с информацией о местоположении магазинов.
6.  **Акции (`/actions/`)** — страница с информацией о товарах на акции.
7.  **Бренды (`/ishop/brands/`)** — страница с брендами товаров.
8.  **Карзина (`/cart/`)** — карзина покупок.
9.  **Список жклаемого (`/profile/wishlist/`)** — список желаемого.

## Технологии
*   **Python 3.13.7**
*   **Django 5.0.2**
*   **SQLite3** (базовая БД для разработки)

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка_репозиторий>
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv .venv
    # Для Windows:
    venv\Scripts\activate
    # Для macOS/Linux:
    source venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Выполните миграции:**
    ```bash
    python manage.py migrate
    ```

5.  **Запустите сервер разработки:**
    ```bash
    python manage.py runserver
    ```
    После запуска сайт будет доступен по адресу: `http://127.0.0.1`
